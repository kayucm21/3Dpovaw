## 2026-05-04 - Глобальное обновление v4.0 — Подписки VLESS и Тюнинг Скорости

### ✨ Новые функции v4.0

#### 🔗 Автообновляемые подписки VLESS (ГЛАВНОЕ!)
- ✅ **Ссылка подписки** - `https://domain.com/sub/token` для автообновления в приложениях
- ✅ **QR-код подписки** - сканирование в приложении для быстрого импорта
- ✅ **Автообновление** - при добавлении/удалении пользователей подписка обновляется автоматически
- ✅ **Поддержка клиентов** - Karing, v2rayNG, Shadowrocket, Quantumult X, Clash Meta
- ✅ **Токен доступа** - уникальный токен для каждой подписки
- ✅ **Срок действия** - настройка времени жизни подписки (7/30/90/365 дней)
- ✅ **Лимит трафика** - ограничение по трафику на подписку
- ✅ **Статистика** - мониторинг использованного трафика

#### ⚡ Тюнинг VLESS для максимальной скорости
- ✅ **Буфер 50MB** - увеличенный буфер для высокой пропускной способности
- ✅ **TCP Fast Open** - снижение задержки при подключении
- ✅ **MUX Concurrency** - 8 одновременных потоков для стабильности
- ✅ **Read/Write Buffer** - оптимизированные буферы 52428800 байт
- ✅ **BBR оптимизация** - улучшенный алгоритм TCP от Google
- ✅ **WebSocket оптимизация** - настроенные параметры WS для стабильности

#### 📊 Мониторинг трафика (НОВОЕ!)
- ✅ **Учёт трафика** - подсчёт потреблённого трафика по пользователям
- ✅ **Графики** - визуализация трафика за 24 часа / 7 дней / 30 дней
- ✅ **Лимиты** - установка лимитов на пользователя
- ✅ **Уведомления** - предупреждение о достижении лимита

#### 🎨 Улучшения UI/UX
- ✅ **Страница подписок** - управление всеми подписками
- ✅ **QR-коды** - генерация QR-кодов для подписок
- ✅ **Копирование ссылок** - быстрое копирование ссылок подписки
- ✅ **Статус подписки** - визуальное отображение статуса (активна/истекла)

### 🛠️ Технические улучшения

#### Backend (server/index.js)
- ✅ `GET /api/subscriptions` - список всех подписок
- ✅ `POST /api/subscriptions` - создание новой подписки
- ✅ `GET /api/subscription/:token` - получение подписки (для клиентов)
- ✅ `DELETE /api/subscriptions/:id` - удаление подписки
- ✅ `POST /api/subscriptions/renew` - продление подписки
- ✅ `GET /api/traffic/:userId` - статистика трафика пользователя

#### Frontend (public/js/app.js)
- ✅ `loadSubscriptions()` - загрузка страницы подписок

- ✅ `copySubscriptionLink()` - копирование ссылки
- ✅ `generateQRCode()` - генерация QR-кода подписки

#### Скрипты установки

#### Новые файлы данных

### ⚙️ Установка v4.0
```bash
cd /opt/naiveproxy-panel
git pull

npm install
pm2 restart naiveproxy-panel
```

### 🔧 Миграция с v3.0
1. Сделайте бэкап через панель
2. Обновите файлы: `git pull`
3. Установите зависимости: `npm install`
4. Перезапустите: `pm2 restart naiveproxy-panel`
5. Создайте подписки для существующих пользователей
